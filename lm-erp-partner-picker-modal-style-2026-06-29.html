UI 통일 완료

거래처원장 검색 모달을 새 판매전표 거래처 선택 모달 형태로 변경

거래선별 원장과 거래 맞춤 화면의 거래처 선택 모달을 새 판매전표의 거래처 선택 모달과 같은 형태로 맞췄습니다.

적용 내용

  • 제목을 거래처 선택으로 통일했습니다.
  • 안내문을 이름 일부를 입력해 검색하세요. ↑/↓ 로 이동, Enter 로 선택, Esc 로 닫기.로 통일했습니다.
  • 테이블형 목록을 새 판매전표와 같은 리스트형 목록으로 변경했습니다.
  • 후보 행에 대표자명, 사업자번호, 전화번호 힌트가 있으면 함께 표시되도록 했습니다.
  • 하단에는 새 판매전표처럼 현재 필터 결과 건수를 표시합니다.
  • 키보드 이동, Enter 선택, Esc 닫기, 클릭 선택 동작은 유지했습니다.
