LM-ERP 거래처 대표자 표시 불일치 수정 보고

문제

(주)서진베아링상사 거래처 화면에서 회사 정보 폼에는 대표자명이 이승준으로 표시되지만, 상단 요약 카드에는 이건면으로 표시되는 문제가 있었습니다.

원인

동일 사업자번호의 고객/공급자 마스터가 여러 개 존재하고 일부 중복 레코드에 과거 대표자명 이건면이 남아 있었습니다. 상단 요약 카드와 입력 폼이 서로 다른 레코드/우선순위에서 값을 가져와 대표자명이 불일치했습니다.

수정

  • 거래처 표시값을 하나의 공통 resolver로 통합했습니다.
  • 상단 요약 카드와 회사 정보 폼이 같은 대표자명/사업자번호/전화번호 값을 사용하도록 수정했습니다.
  • 공백 값은 유효값으로 보지 않고 다른 역할의 값으로 fallback하도록 보강했습니다.
  • DB 데이터는 임의 수정하지 않았습니다.

  • 통과 전체 테스트 709개
  • 통과 npx tsc --noEmit --pretty false
  • 통과 vercel build --prod
