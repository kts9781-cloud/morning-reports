4월말 이관 미수 누락분 직접 입력 기능

성진(양평) 등 현재 선택한 사업장에서 엑셀 이관 시 누락된 4월말 기준 미수 잔액을 거래처별로 직접 입력할 수 있도록 기능을 추가했습니다.

사용 위치

  • /accounts/receivable/import — 기초 채권 이관 화면 상단의 누락분 직접 입력 섹션

입력 방식

  • 기준월을 2026-04로 선택합니다.
  • 거래처명/코드 일부를 입력합니다. 예: 조이테크 또는 일부 검색어.
  • 이관 누락 금액을 입력합니다. 예: 307120.
  • ADD: 기존 기준월 이관 잔액에 더합니다. 기본값입니다.
  • REPLACE: 기준월 이관 잔액을 입력 금액으로 덮어씁니다.

처리 기준

  • 현재 선택된 사업장 기준으로만 거래처를 찾고 저장합니다.
  • 고객 거래처가 없고 같은 사업장 매입처가 있으면 매출처 역할을 생성한 뒤 저장합니다.
  • 부분 검색 결과가 여러 개면 저장하지 않고 후보를 보여줘 오입력을 막습니다.
  • 미수금현황 계산 기준에 맞게 month01_amount, receivable_total_amount, balance_amount가 함께 반영됩니다.
