매입 체크를 해제했는데도 매입 행이 계속 보이던 문제를 재수정했습니다. 이제 체크 해제 즉시 해당 종류의 행을 화면에서 바로 숨깁니다.

문제 재확인

이전 수정은 체크박스 클릭 시 kind=sale / kind=purchase URL로 이동하게 만들었지만, 서버 재조회가 끝나기 전 기존 rows가 화면에 남았습니다. 그래서 매입 표시가 꺼진 상태에서도 매입 행이 그대로 보이는 현상이 발생했습니다.

이번 재수정 핵심

완료체크 즉시 화면 필터

서버 재조회 완료를 기다리지 않고, 클라이언트 테이블에서 현재 체크 상태 기준으로 visibleRows를 만들어 렌더링합니다.

완료매입 행 즉시 숨김

매입 체크가 꺼지면 direction='purchase', 현재 매입 전표, 매입 VAT, 지급 행을 즉시 숨깁니다.

완료매출 행 즉시 숨김

매출 체크가 꺼지면 direction='sale', 수금, 매출 VAT 행을 즉시 숨깁니다.

완료URL 필터 유지

화면 즉시 필터와 별개로 기존처럼 kind=sale / kind=purchase URL 이동도 유지해 서버 데이터도 동기화됩니다.
