LM-ERP 거래선별 원장 VAT 세금계산서 내역 링크 추가 보고

요청

거래선별 원장에서 부가세(VAT) 금액을 클릭하면 해당 월의 세금계산서 발행 내역으로 이동하도록 개선했습니다.

수정

  • 매출 VAT 행은 /invoices?tab=sales&month=YYYY-MM으로 이동합니다.
  • 매입 VAT 행은 /invoices?tab=purchase&month=YYYY-MM으로 이동합니다.
  • 예: 2026-05-31 / 부가세 / VAT / 226,011 클릭 시 /invoices?tab=sales&month=2026-05로 이동합니다.
  • VAT 행의 금액만 파란색 링크로 표시하고, 일반 행/일계/월계는 기존 표시를 유지합니다.

검증

  • 통과 legacyVatDeepLink.test.ts 11개 테스트
  • 통과 npx tsc --noEmit --pretty false
  • 통과 vercel build --prod
  • 완료 운영 prebuilt 배포
